# BooksExchange 📚

BooksExchange is a React-based book exchange platform that allows users to add, browse, and discover books while earning points.  
The app uses **dummy data (localStorage)** to simulate backend behavior, making it fully functional without a real database.

---

## ✨ Features

- Add and browse books
- Search, filter, and sort books
- Points system based on book condition
- Book history tracking
- QR payload generation (for future scanning)
- OpenStreetMap integration for exchange locations
- Dummy authentication (Login / Signup)
- Works without a backend (localStorage-based)

---

## 🛠 Tech Stack

- React (Vite)
- React Router v6
- Leaflet + OpenStreetMap
- LocalStorage (dummy data)
- CSS-in-JS (inline styles)
